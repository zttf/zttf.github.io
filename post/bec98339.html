3.容器命令

1.新建+启动容器

docker run -it centos /bin/bash

2.列出当前正在运行的容器

docker ps docker ps -a 列出历史容器

3.退出容器

exit —-> run进去，exit退出，容器停止

ctrl+p+q —->run进去，ctrl+p+q退出，容器不停止

4.重启容器

docker restart mysql

5.删除容器

docker rm mysql

docker rm -f mysql 强制删除
